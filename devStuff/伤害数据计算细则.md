# 异常状态数学期望
- 计算公式：每批子弹的数量（Fabricate amount） * probably * 该异常状态强度（Affliction amount） = 其数学期望
- 以下标签均属于异常状态：
    sufforinpoisoning、morbusinepoisoning、cyanidepoisoning、huskinfection这些标签的数学期望应当控制在不超过20
    incrementalstun 的数学期望不超过300
    opiateoverdose的数学期望不超过50
    opiateaddiction的数学期望不超过25
- 一般来讲，不需要对probably做出修改（默认为1），只有在amount值计算后超出1，才需要使用数学期望将其转换为对应的概率，
- 生成代码时，同一个弹种的数量是规定好的——见每个弹种提供的模板，不允许修改

# 枪伤
gunshotwound、bleeding、organdamage均视为枪伤，一般来讲，不需要计算其数学期望，将amount提升即可，如有probably则保留
